# 📊 Stock Portfolio Tracker

## 📌 Overview

This project is a simple **Stock Portfolio Tracker** built using Python.
It allows users to:

* Enter stock names
* Input quantity
* Calculate total investment
* Save results to a file

---

## 🛠️ Technologies Used

* **Python 3**
* File Handling

## ▶️ How to Run

1. Open project in **VS Code**
2. Open terminal
3. Run the script:

```bash
python StockTracker.py
```

---

## 💻 How It Works

1. The program contains predefined stock prices:

   * AAPL → 180
   * TSLA → 250
   * GOOG → 140
   * AMZN → 130

2. User inputs:

   * Stock name
   * Quantity

3. Program calculates:

   * Individual investment
   * Total investment

4. Final result:

   * Displays total investment
   * Saves it in **portfolio.txt**
